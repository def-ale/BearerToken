# GitHub API com Express

Este projeto demonstra como acessar a API do GitHub utilizando o Express e o token de acesso pessoal.

## Nome da API
GitHub API

## URL da API
https://api.github.com

## Documentação
[GitHub API Docs](https://docs.github.com/en/rest)

- Node.js
- NPM (ou Yarn)
